About API

If you want to browse the API from any other platform, below urls can be used for CRUD operation:
1. Genre API: https://demo-mysiteapi.herokuapp.com/api/genre/
2. Movie API: https://demo-mysiteapi.herokuapp.com/api/movie/

Note: Genres provided in Movie model should be present in Genre model. If you want to add a new genre which is not present in Genre Model. Then first add that to Genre then use in Movie object creation.
